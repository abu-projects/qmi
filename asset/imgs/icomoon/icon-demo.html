Icon Font Demo

Font Generation Required

The icons on this page are placeholders. To see the actual icons, you need to generate a font from your SVG files.

Please follow the instructions in fonts/FONT_GENERATION_GUIDE.md to generate the icon font.

Once you've generated the font files, place them in the fonts directory and the icons will appear correctly.

This page demonstrates the icon font system and provides a cheat sheet for all available icons.
